# Deck memorizer 

```
Usage:
  deck-memorizer [OPTIONS] memorize [memorize-OPTIONS]

Help Options:
  -h, --help                   Show this help message

[memorize command options]
      -s, --max-deck-size=     Maximum number of cards
          --show-associations  Show associations
          --select=            select card types. D for diamond, H for heart
          --associations-file= path to the associations map file
```
